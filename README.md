# ПРОЕКТ КНИГА — Telegram-бот для усвоения книг

Бот помогает глубже понимать сложные книги через открытые вопросы после каждой главы. Первая книга: «Почему бедные страны беднеют, а богатые страны богатеют» (Эрик Райнерт).

## Как работает
1. Читаешь главу
2. `/start` в боте → выбираешь главу → бот задаёт 3–7 открытых вопросов
3. Отвечаешь своими словами
4. Бот оценивает по ключевым словам (с учётом склонений — pymorphy3)
5. Средний балл ≥ 1.5 → следующая глава; < 1.5 → перечитать

## Стек
- Python 3.11+
- aiogram 3.x
- SQLite (прогресс)
- JSON (главы + вопросы)
- pymorphy3 (лемматизация русского)
- OpenAI API — опционально, выключено по умолчанию (бесплатный режим)

## Быстрый старт (3 команды)
1. Установи Python 3.11+ с галочкой «Add to PATH»
2. `setup.bat` — поставит зависимости
3. `run.bat` — запустит бота

Полная инструкция — в `УСТАНОВКА.md`.

## Добавить новые главы / вопросы
Открой `data/chapters.json`. Для каждого вопроса заполни:
- `text` — сам вопрос
- `reference_answer` — эталон (нужен если когда-нибудь включишь ИИ-режим)
- `keywords` — список ключевых слов/фраз, которые должны быть в ответе
- `required_count` — сколько из них достаточно для 2 баллов

## Команды бота
- `/start` — приветствие, список книг
- `/stats` — пройдено глав и средний балл
- `/reset` — сброс прогресса (только для ADMIN_USER_ID)
