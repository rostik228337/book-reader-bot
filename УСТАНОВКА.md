# Установка и запуск — пошагово

Всё делается один раз. Дальше запуск бота — одна команда.

## ШАГ 1. Установить Python 3.11+

### Windows
1. Скачай с https://www.python.org/downloads/
2. При установке **обязательно поставь галочку "Add Python to PATH"**
3. Проверка — открой PowerShell и введи:
   ```
   python --version
   ```
   Должно показать `Python 3.11.x` или выше.

## ШАГ 2. Получить токен Telegram-бота

1. Открой Telegram, найди бота **@BotFather**
2. Напиши ему `/newbot`
3. Придумай имя (любое, например `Книга Райнерта бот`)
4. Придумай username (должен кончаться на `bot`, например `reinert_book_bot`)
5. BotFather пришлёт строку вида `123456789:AAH...` — **это твой `BOT_TOKEN`, сохрани его**

## ШАГ 3. Узнать свой Telegram ID

1. Найди в Telegram бота **@userinfobot**
2. Нажми `/start`
3. Он покажет твой `Id` — число вида `123456789`. **Сохрани — это `ADMIN_USER_ID`**

## ШАГ 4. (Опционально) Получить OpenAI API ключ

Нужен только если хочешь включить умную оценку ответов через ИИ. **Для MVP можно пропустить.**

1. Зайди на https://platform.openai.com/api-keys
2. Нажми "Create new secret key" → скопируй (`sk-...`)
3. Пополни баланс хотя бы на $5 (оценка одного ответа = доли цента на gpt-4o-mini)

## ШАГ 5. Сгенерировать код бота

1. Открой файл **`ПРОМТ_ДЛЯ_КОДА.md`** в этой папке
2. Скопируй **весь его текст целиком**
3. Открой любой инструмент:
   - **Cursor** (рекомендую, https://cursor.com)
   - или Claude Code
   - или ChatGPT / Claude в браузере
4. Вставь промт → нажми отправить
5. ИИ сгенерирует файлы — сохрани их в папку `bot/` (если используешь Cursor — открой в нём папку `ПРОЕКТ КНИГА`, он сам разложит)

После генерации в папке `bot/` должны появиться:
- `main.py` — основной файл бота
- `grader.py` — модуль оценки ответов
- `storage.py` — работа с SQLite
- `requirements.txt` — список зависимостей
- `.env.example` — шаблон конфигурации

## ШАГ 6. Заполнить `.env`

1. Скопируй `bot/.env.example` → `bot/.env`
2. Открой `bot/.env` в блокноте и впиши значения:
   ```
   BOT_TOKEN=123456789:AAH...твой токен от BotFather
   ADMIN_USER_ID=123456789
   USE_AI_GRADING=false
   OPENAI_API_KEY=
   ```

## ШАГ 7. Установить зависимости

В PowerShell (в папке проекта):
```powershell
cd "C:\Users\User\Documents\Claude\Projects\ПРОЕКТ КНИГА\bot"
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## ШАГ 8. Наполнить первую главу

Открой `data/chapters.json` → там уже будет шаблон с главой 1 для примера.
Заполни **настоящие вопросы и ключевые слова** по Главе 1 книги Райнерта. Подробнее — см. сам файл `chapters.json` с комментариями внутри.

## ШАГ 9. Запустить бота

```powershell
cd "C:\Users\User\Documents\Claude\Projects\ПРОЕКТ КНИГА\bot"
venv\Scripts\activate
python main.py
```

Бот запущен! Открой Telegram, найди своего бота по username, нажми `/start`.

## Дальше

- Чтобы остановить бота — в PowerShell нажми `Ctrl+C`
- Чтобы запустить снова — повтори ШАГ 9
- Хочешь включить ИИ-оценку — поменяй в `.env`: `USE_AI_GRADING=true` и впиши `OPENAI_API_KEY`, перезапусти бота

## Что устанавливаем где — кратко

| Что | Куда | Зачем |
|---|---|---|
| Python 3.11+ | На компьютер глобально | Чтобы запускать бота |
| aiogram, pymorphy3, openai | В папку `bot/venv/` (автоматически через pip) | Библиотеки бота |
| BOT_TOKEN | В `bot/.env` | Идентификация бота в Telegram |
| OPENAI_API_KEY (опц.) | В `bot/.env` | Для ИИ-оценки |
| Главы и вопросы | В `data/chapters.json` | Контент |
| Прогресс | В `data/progress.db` (создаётся автоматически) | История ответов |
